## 2025-03 — 서비스 방향 결정

### ADR-001 단순 PPT 도구 → 예배 준비 올인원 플랫폼

**결정:** 성경봉독 PPT 생성기를 넘어서, 설교 준비 보조 + 예배 PPT 자동화 통합 서비스로 방향 확장

**이유:**
- 단순 PPT 도구는 기존 Bible2PPT와 차별점 없음
- 성경 구절 중심으로 설교 방향·찬양 선택까지 연결하면 실질적 가치 창출
- 국내에 통합 솔루션 부재 — 틈새시장

**영향:** 로그인/교회 계정, 템플릿, AI 기능 추가 필요

## 2025-03 — 아키텍처 결정

### ADR-002 PPT 병합을 백엔드에서 처리

**결정:** Apache POI (Spring Boot)에서 PPT 병합

**이유:**
- 사용자가 기존 .pptx 파일을 첨부하는 기능 때문에 브라우저(PptxGenJS)만으로 불가
- 기존 파일 읽기·슬라이드 추출·병합은 서버사이드에서만 안정적으로 처리 가능

**트레이드오프:**
- 백엔드 메모리 사용량 증가 (대용량 파일 처리 시)
- 운영 안정화 후 Aspose.Slides 도입 검토 예정 (Apache POI보다 병합 안정성 높음)

**영향:** `ppt/` 패키지 추가, 파일 스토리지 필요, 프론트는 최종 다운로드만 담당

### ADR-003 슬라이드 생성 방식 이원화 (AUTO / FILE)

**결정:** 모든 예배 항목에서 자동 생성(AUTO)과 파일 첨부(FILE) 중 선택 가능

**이유:**
- 교회마다 수년에 걸쳐 다듬어온 PPT 자산이 있음 (찬양 악보, 교독문 일러스트 등)
- 코드로 재현하면 퀄리티가 기존 파일보다 낮아질 수밖에 없음
- 기존 파일 퀄리티를 유지하면서 수고(복사·붙여넣기)만 제거하는 전략이 현실적

**영향:** `WorshipItem.mode` 컬럼 (AUTO/FILE), `UploadedFile` 테이블 추가

---

### ADR-004 예배 순서 템플릿 설계

**결정:** 템플릿은 "순서 구조만" 저장. PPT 디자인 템플릿은 별도 관리하지 않음

**이유:**
- 교회마다 예배 순서가 다르므로 템플릿 저장·재사용은 필요
- PPT 디자인은 파일 첨부(FILE 모드)로 해결 → 별도 디자인 템플릿 불필요
- AI로 예배 순서 추천은 하지 않음 — 순서는 교회 고유의 것

**영향:** `worship_template` + `template_item` 테이블, AI는 콘텐츠 추천에만 집중

## 2025-03 — 기술 스택 결정

### ADR-005 AI 모델을 OpenAI GPT-4o로 선택

**결정:** Claude API 대신 OpenAI GPT-4o 사용 (초기)

**이유:**
- 설교 개요·찬송가 추천·연관구절 같은 범용 추천은 GPT-4o가 비용 효율적
- Phase 3 설교문 초안(깊은 신학적 맥락) 단계에서 모델 재비교 후 결정

**영향:** `ai/client/OpenAiApiClient.java`, 환경변수 `OPENAI_API_KEY`

### ADR-006 성경 데이터를 갓피플 스크래핑으로 처리

**결정:** 개역개정 성경 데이터를 갓피플에서 런타임 스크래핑 + Redis 캐싱

**이유:**
- 개역개정은 대한성서공회 저작권 — JSON 번들 배포 시 저작권 침해
- 갓피플 스크래핑은 Bible2PPT와 동일한 방식, 비상업적 교회 용도로 실제 문제된 사례 없음
- Redis 24시간 캐싱으로 반복 요청 최소화

**트레이드오프:** 갓피플 HTML 구조 변경 시 셀렉터 재조정 필요

**셀렉터 (2025-03 확인):**
- 절 번호: `td.bidx_listTd_yak` → `ownText()`
- 본문: `td.bidx_listTd_phrase span.line_pen_ > span.line_pen_` → `ownText()`

---

### ADR-007 교독문 데이터를 JSON 번들로 처리

**결정:** 장로교 교독문 1~184번을 `resources/data/responsive-reading.json`으로 번들

**이유:**
- 찬송가·성경과 달리 교독문은 공개 텍스트로 활용 가능
- 런타임 스크래핑 불필요 → 안정적
- 기본 교단: 장로교 (통합/합동), 추후 감리교·침례교 등 확장 예정
